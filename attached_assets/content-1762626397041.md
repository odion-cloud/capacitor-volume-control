## About

[Permalink: About](https://github.com/capacitor-community/volume-buttons#about)

This plugins allows to listen for the events fired when the user presses the hardware volume up or down button of the device. An object that contains only one property is passed to the callback - see [VolumeButtonsResult](https://github.com/capacitor-community/volume-buttons#VolumeButtonsResult).
This plugin contains code derived from or inspired by [https://github.com/CipherBitCorp/VolumeButtonHandler](https://github.com/CipherBitCorp/VolumeButtonHandler) and [https://github.com/thiagobrez/capacitor-volume-buttons](https://github.com/thiagobrez/capacitor-volume-buttons) plugins.

**Features:**

- support receiving events when the volume is max or min (make sure to use **`disableSystemVolumeHandler`** on iOS platform - see [VolumeButtonsOptions](https://github.com/capacitor-community/volume-buttons#VolumeButtonsOptions))
- keep receiving events after the application sent to and returns from the background
- supports Android and iOS platforms

**NOTE**: The plugin version 7.0.0 is compatible with Capacitor 7

### Add volume button listener in the app

[Permalink: Add volume button listener in the app](https://github.com/capacitor-community/volume-buttons#add-volume-button-listener-in-the-app)

```
import { VolumeButtons } from '@capacitor-community/volume-buttons';

const options: VolumeButtonsOptions = {};
const callback: VolumeButtonsCallback = (result: VolumeButtonsResult, err?: any) => {
  console.log('result', result);
};
if (this.platform.is('ios')) {
  options.disableSystemVolumeHandler = true;
} else if (this.platform.is('android')) {
  options.suppressVolumeIndicator = true;
}
await VolumeButtons.watchVolume(options, callback);
```

### Remove volume button listener in the app

[Permalink: Remove volume button listener in the app](https://github.com/capacitor-community/volume-buttons#remove-volume-button-listener-in-the-app)

```
import { VolumeButtons } from '@capacitor-community/volume-buttons';

await VolumeButtons.clearWatch();
```
